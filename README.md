# Calculator Project (Python)

This project shows my journey in programming — from building my **first Python calculator** to developing a more structured and advanced version with a graphical interface.

---

##  Overview

This repository contains multiple versions of a calculator over the years, each representing an improvement in my coding skills and understanding of programming concepts.

---

##  Version 1 — Basic Calculator (Grade 5)

**Features:**

* Performs basic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Uses simple functions and user input

**Concepts Learned:**

* Functions
* Conditional statements (`if/elif`)
* User input handling

---

## Version 2 — Improved Calculator

**Features:**

* Continuous calculations (loop-based)
* Error handling (invalid input, division by zero)
* Cleaner and modular code structure

**Concepts Learned:**

* Exception handling (`try/except`)
* Code reusability (functions)
* Better program flow design

---

## Version 3 — GUI Calculator

**Features:**

* Graphical User Interface using Tkinter
* Button-based input system
* Real-time calculations

**Concepts Learned:**

* GUI development with Tkinter
* Event-driven programming
* User interface design basics

---

## Technologies Used

* Python
* Tkinter (for GUI)

This was one of my earliest projects, originally created during my early learning phase.
